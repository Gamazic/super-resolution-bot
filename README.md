# super-resolution-bot

## Bot link

https://t.me/GamazicSuperResolutionBot

## Start localy via docker-compose

### Requirements
* `docker-compose`

### Actions

* Create `.env` file with folowing variables
    ```
    TG_TOKEN
    ```
* Run the following command
    ```bash
    docker-compose up -d
    ```
* Wait for the end of the build and use the bot
